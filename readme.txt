Responsive   Carousel Slider  for online-shops with WooCommerce.

== Description ==

Slider with the possibility of withdraw the goods by the ID  of products. <br> Just pointing id you will quickly fill the  slider, <br> and also you will be able to  insert it into any page of your site very fast.

== Installation ==

1. Download the plugin and install it
2. Activate the plugin
3. Go to the tab AVED slider option in WordPress console
4. Fill the slides of  products' ID
5. Insert the shortcode to the needed page 

== Frequently Asked Questions ==

1. How to add a slider on the page? <br> * Insert the shortcode "[show_SAS]" <br>
2. How to add new slides? <br> *  On the top right there is a   "ADDslide" button <br>
3. How to delete a slide? <br> *  On the right of the slide there is  "Delete" button (ID input field ) <br>
4. How to save the settings? <br> * On the top right there is a "Save" button <br>
5. Is it possible to create different sliders? <br> * Not yet, but  it is expected in <br> updates
